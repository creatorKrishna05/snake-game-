# 🐍 Snake Water Gun Game

A simple command-line Snake Water Gun game built using Python. In this game, you play against the computer. The computer randomly selects Snake, Water, or Gun, and the winner is determined according to the game rules.

---

## 🎮 Game Rules

- 🐍 Snake drinks Water → Snake Wins
- 💧 Water douses Gun → Water Wins
- 🔫 Gun kills Snake → Gun Wins
- Same choice → It's a Draw

---

## 🚀 Features

- Play against the computer
- Random computer choice
- Win and lose sound effects
- Simple command-line interface
- Beginner-friendly Python project
